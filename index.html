Research Projects (IIT Kanpur)

  • LLM-Based PISA Questions (Sawaal Machine): Designed and implemented an LLM-based pipeline, deployed as the tool Sawaal Machine, to generate assessment items aligned with PISA standards across Math, Science, and Reading domains.[cite: 2] Built domain-specific question banks and randomized test generation tools for scalable assessment design.[cite: 2]
  • Bridging the Gap Between Cognitive Science And Neural Network: Common Sense: Designed and Developed a Hopfield network model from scratch, achieving over 70% accuracy on the MNIST handwritten digit classification benchmark using basic numpy and pandas libraries.[cite: 2]
  • RAG in LLM: Implemented RAG to generate new data interpretation questions using openai API via creating a separate knowledge base for data interpretation questions.[cite: 2] Performed A/B Testing in order to compare human vs. AI generated data interpretation questions in Psychopy.[cite: 2]
  • Human-Computer Interaction (CS798H): Redesigned the ECI Voter Portal using Gestalt principles to enhance usability, visual clarity, and mobile responsiveness.[cite: 2] Conducted thematic analysis of 30+ Amazon Alexa user reviews to identify key UX issues.[cite: 2] Co-developed Hot Desker, a full-stack web-based seat booking system for hybrid workplaces.[cite: 2]
