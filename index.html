Research Projects

  • LLM-Based PISA Questions (Sawaal Machine): Designed and implemented an LLM-based pipeline, deployed as the tool Sawaal Machine, to generate assessment items aligned with PISA standards across Math, Science, and Reading domains. Built domain-specific question banks and randomized test generation tools for scalable assessment design.
  • Bridging the Gap Between Cognitive Science And Neural Network: Common Sense: Designed and Developed a Hopfield network model from scratch, achieving over 70% accuracy on the MNIST handwritten digit classification benchmark using basic numpy and pandas libraries.
  • RAG in LLM: Implemented RAG to generate new data interpretation questions using openai API via creating a separate knowledge base for data interpretation questions. Performed A/B Testing in order to compare human vs. AI generated data interpretation questions in Psychopy.
  • Human-Computer Interaction (CS798H): Redesigned the ECI Voter Portal using Gestalt principles to enhance usability, visual clarity, and mobile responsiveness. Conducted thematic analysis of 30+ Amazon Alexa user reviews to identify key UX issues. Co-developed Hot Desker, a full-stack web-based seat booking system for hybrid workplaces.
